# Épico 5: Engajamento

**Descrição:** Engloba a camada de gamificação do sistema. É responsável por contabilizar as interações positivas dos veteranos, gerenciar o sistema de pontuação/conquistas e renderizar os painéis de reconhecimento da comunidade.

**Requisitos Funcionais Associados:** * RF16, RF17

---

## Histórias de Usuário (User Stories)

* **US5.1:** Como veterano, quero acumular pontos e obter medalhas (badges) digitais ao responder a dúvidas no fórum ou cadastrar informações úteis, validando meu apoio à comunidade.
* **US5.2:** Como usuário da plataforma, quero visualizar um ranking com os veteranos mais ativos para reconhecer e identificar os estudantes que mais contribuem para o ecossistema da universidade.
