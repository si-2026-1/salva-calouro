# Épico 4: Informação e Comunidade

**Descrição:** Responsável pela centralização de dados burocráticos e pela interação social. Reúne a base de conhecimento estática (FAQ), o calendário de prazos da instituição e o fórum colaborativo de perguntas e respostas.

**Requisitos Funcionais Associados:** RF13, RF14, RF15

---

## Histórias de Usuário (User Stories)

* **US4.1:** Como calouro com dúvidas institucionais, quero consultar uma base de conhecimento (FAQ) com pesquisa otimizada para entender rapidamente processos como matrícula, trancamento e ajustes.
* **US4.2:** Como veterano, quero criar tópicos e responder a dúvidas no fórum da plataforma para compartilhar minha experiência e ajudar na integração dos novos alunos.
* **US4.3:** Como usuário, quero visualizar as datas críticas e os prazos importantes do calendário acadêmico para não perder os períodos oficiais de modificação de matrícula.
