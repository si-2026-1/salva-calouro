# Épico 1: Gestão de Usuários

**Descrição:** Este épico abrange o fluxo de entrada, autenticação e controle de acesso dos usuários na plataforma, garantindo a diferenciação funcional e de permissões entre os perfis que consomem informação e os que a geram.

**Requisitos Funcionais Associados:** * RF01, RF02, RF03, RF04

---

## Histórias de Usuário (User Stories)

* **US1.1:** Como usuário, quero me cadastrar na plataforma para poder acessar as funcionalidades personalizadas de acordo com o meu curso.
* **US1.2:** Como usuário cadastrado, quero fazer o login de forma segura para acessar minhas preferências, histórico de fluxo e dados salvos.
* **US1.3:** Como estudante, quero que o sistema identifique se sou calouro ou veterano para que a interface exiba as ferramentas adequadas ao meu nível de permissão.
* **US1.4:** Como usuário, quero poder editar as informações do meu perfil para manter meus dados acadêmicos e de contato atualizados.
